    • Introducción a las Colecciones

      Las colecciones son una manera muy fácil de gestionar tu librería de iconos y el núcleo de tu aplicación web. ¿Quieres saber cómo usarlas? Presiona la flecha de la derecha o use la navegación superior para ver la guía.

    • Añadir iconos

      Explora los packs o busca iconos y añádelos a tu colección activa con este botón.

    • Vista lista

      Esta es una lista de tus colecciones. Puedes cambiar sus nombres y están ordenados por uso. Haz clic en una de ellas para empezar a usarla.

    • Crea una colección

      Haz clic en este botón para crear una nueva colección.

    • Activa colección

      Esta es tu colección activa. Una colección puede tener hasta 256 iconos si eres un usuario registrado, o solo 50 si no estás registrado.

    • Opciones de colecciones

      Accede a este menú para editar tu colección; edita tus iconos separadamente, duplica tu colección, ordena los iconos o comparte tu colección con el resto del mundo.

    • Subida SVG

      Arrastra tus SVG a la colección activa para subir tus propios iconos. Hay una límite de 3 iconos por colección para usuarios gratuitos.

    • Pantalla completa

      NEW

      ¿No tienes suficiente espacio? Ahora puedes ver tus colecciones en plantalla completa para facilitar su uso.

    • Eliminar

      ¿Ya no necesitas más esta colección? Bórrala y crea una nueva aquí.

    • ¡Disfruta!

      Y eso es todo. Si tienes alguna pregunta, puedes contactar con nuestro soporte aquí.
